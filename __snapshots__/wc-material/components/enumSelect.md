# `wc-material/components/enumSelect`

#### `renders an mwc-select`

```html
<mwc-select>
  <mwc-list-item value="foo">
    foo
  </mwc-list-item>
  <mwc-list-item value="bar">
    bar
  </mwc-list-item>
</mwc-select>

```

#### `sets selection to current object`

```html
<mwc-select>
  <mwc-list-item value="foo">
    foo
  </mwc-list-item>
  <mwc-list-item
    selected=""
    value="bar"
  >
    bar
  </mwc-list-item>
</mwc-select>

```
